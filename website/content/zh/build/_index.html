使用 Operator SDK 来构建 Operators

我怎样才能使用 Operator SDK 来编写一个 Operator 呢？

使用 SDK 有很多种方式来构建 Operator。如果想要现在就着手操作，请参参考此快速入门中的步骤。

安装 SDK CLI

按照安装指南中所给的步骤学习如何安装 Operator SDK CLI 工具。如果你在使用 SDK 的某个发行版本，请确保遵循该版本的文档。你可以使用以下几种方式来安装：

  1. 使用 Homebrew 安装 (macOS)
  2. 基于 GitHub 发行页面安装
  3. 使用 master 分支代码编译并安装
