Operators 🔗 被用来在 Kubernetes 上管理复杂的又状态应用。 Operator SDK 通过提供高层 API、有用的抽象概念以及项目结构生成等能力，使得构造 Operator 的工作变得很容易。

{{< /blocks/cover >}} {{< blocks/lead height="auto" color="white" >}} 此项目是 [Operator Framework](https://github.com/operator-framework) 的一部分， 是一个以高效、自动化且可扩缩方式来管理称作 Operaters 的 Kubernets 原生应用的工具。 进一步了解可阅读[介绍博客](https://coreos.com/blog/introducing-operator-framework). {{< /blocks/lead >}}
